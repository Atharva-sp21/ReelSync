# [ReelSynce] 🚀

An end-to-end automated pipeline that scrapes Instagram Reels, uses AI to categorize the vibe, and pushes the data to a live, searchable public dashboard. Built for [Hackathon Name].

## 🛠️ The Architecture
* **Trigger:** Twilio WhatsApp Bot / n8n Webhook
* **Scraper:** RapidAPI Instagram Scraper (Stable API)
* **AI Processing:** OpenAI via n8n LLM Chain
* **Database:** Airtable
* **Frontend UI:** Softr
